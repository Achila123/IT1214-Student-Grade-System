# IT1214-Student-Grade-System
Student Grade Management System – Java OOP project

## Project Purpose
A console-based Java application to manage student grades using OOP principles. Teachers can:
- Add new student records (ID, name, marks)
- View all records
- Search for a student by ID
- Calculate the average marks of all students

## How to Run the Program
1. Ensure Java JDK (8 or higher) is installed.
2. Open a terminal/command prompt in the project folder.
3. Compile both Java files:
  javac Student.java
  Main.java
